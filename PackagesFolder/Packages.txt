📘 Java Packages

🔹 What is a Package?
Package is a group of similar types of classes, interfaces and sub-packages. Packages can be built-in or user defined.


🔹 Types of Packages

1. Built-in Packages
-Already provided by Java.
-No need to create them.

Examples:-
 java.lang → String, Math, System *(auto imported)*
 java.util → Scanner, ArrayList
 java.io → File handling
 java.sql → Database


2. User-defined Packages
-Created by programmer using package keyword.

Syntax:
package packagename;
